# NFT Preview Card

Implementación del reto **NFT Preview Card Component** de Frontend Mentor.

Proyecto enfocado en maquetación precisa, estructura semántica y CSS limpio.

---

## Objetivo

Reproducir el diseño proporcionado respetando:

- Jerarquía visual  
- Espaciado consistente  
- Buenas prácticas en HTML  
- CSS organizado y mantenible  

---

## Tecnologías

- HTML5  
- CSS3  
- Flexbox    
- Pseudo-elementos (`::before`, `::after`)  

---

## Enfoque

- Estructura semántica clara  
- Uso de `position: relative` como contexto para capas  
- Overlay controlado con `position: absolute` y `z-index`  
- Efectos `hover` implementados únicamente con CSS  
- Uso de unidades relativas (`rem`) para consistencia visual
